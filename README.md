Добро пожаловать в репозиторий модульной системы стрельбы для **Unreal Engine 5.7**! 🎮 Этот проект представляет собой набор высокопроизводительных C++ компонентов, которые позволяют легко реализовать гибкую и масштабируемую механику стрельбы в любом шутере.

---

## 🎯 Основные компоненты

Система построена на принципах компонентной архитектуры (Actor Components), что позволяет использовать их независимо:

- **`ShootingComponent` 🔫**
  - Ядро системы: управляет логикой выстрела, расчетом траектории и темпом стрельбы.
- **`ReloadComponent` 🔋**
  - Отвечает за менеджмент магазина, анимации перезарядки и состояние оружия.
- **`ObjectPoolComponent` ♻️**
  - Обеспечивает высокую производительность за счет переиспользования объектов (снарядов, эффектов).
  - Минимизирует нагрузку на сборщик мусора (Garbage Collector).
- **`UPoolableComponent` ✨**
  - Базовый компонент, который должны наследовать все акторы, участвующие в пуле объектов.
  - Обеспечивает интерфейс для инициализации, сброса и возврата в пул.
  - Взаимодействует с `ObjectPoolComponent` для управления жизненным циклом объекта.
- **`HitImpactComponent` ✨**
  - Автоматически обрабатывает визуальные и звуковые эффекты при попадании в зависимости от типа физической поверхности.

---

## 🛠️ Преимущества архитектуры

- **Модульность:** Добавляйте только те компоненты, которые нужны вашему оружию.
- **Производительность:** Пул объектов позволяет обрабатывать сотни снарядов без падения FPS. ⚡
- **Гибкость:** Полностью настраиваемые параметры через редактор Unreal Engine.
- **Чистый код:** Реализация на C++ обеспечивает максимальную скорость работы и удобство расширения.
